
# Release 2.0.6 (2026-05-19)

## Bug fixes

- **replace stale gh auth token preflight with vrg-gh credential model**
vrg-gh resolves credentials internally via gh auth status/token. The old preflight tried to run gh auth token directly, which is denied by the vrg-gh wrapper. Replace with a lightweight vrg-gh command to verify auth is functional.
